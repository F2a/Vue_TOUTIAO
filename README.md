* 字体图标 iconfont
* CSS扩展 less + normalize
* 模拟接口 easy-mock

* 组件库 mint-ui

### 文件布局
``` bash
|-- build                            // webpack配置文件
|-- config                           // 项目打包路径
|-- src                              // 源码目录
|   |-- common                       // 公共资源目录
|       |-- font                     // iconfont目录
|       |-- style                    // 公共样式文件目录
|   |-- components                   // 组件
|-- static                           // 静态资源目录
```


``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
